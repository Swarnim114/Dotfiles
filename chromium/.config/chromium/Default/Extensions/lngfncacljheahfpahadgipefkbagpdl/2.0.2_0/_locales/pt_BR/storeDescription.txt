Experimente a verdadeira visualização cinematográfica no seu monitor 21:9 com o UltraWide Video! Esta extensão transforma o modo de tela cheia no YouTube, Netflix, HBO GO e outras plataformas de streaming, garantindo uma proporção de tela ultra-wide envolvente.

Principais Recursos:
 • Suporte Aprimorado para Ultra-Wide: Ajusta automaticamente a proporção de tela para monitores ultra-wide (21:9, 32:9), eliminando barras pretas nos vídeos.
 • Alternância Instantânea: Troque facilmente a proporção de tela com um simples atalho:
     • Windows: Pressione Ctrl+Alt+C
     • macOS: Pressione Ctrl+Cmd+C
 • Uso Versátil: Perfeito para vídeos com proporção de tela larga que não escalam corretamente, garantindo uma experiência de visualização perfeita.
